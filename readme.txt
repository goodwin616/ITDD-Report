Aspose.Words for Java is an advanced class library for Java that enables you to perform 
a great range of document processing tasks directly within your Java applications.

Aspose.Words supports DOC, OOXML, RTF, HTML, OpenDocument, PDF, XPS, EPUB and other formats. 
With Aspose.Words you can generate, modify, convert and render documents without using Microsoft Word.

For more info see http://www.aspose.com/categories/java-components/aspose.words-for-java/default.aspx
